CyberFlex provides an iOS experience for cyber security education, through an introduction into Capture-The-Flag activities. With 10 flags to find, can you work through the challenges and identify the flags? You'll learn about file permission issues, web security issues, and operating system issues - all valuable lessons as part of developing cyber security awareness and knowledge.

This provides a fun and interactive platform for developing skills, and will be extended for other cyber security challenges.

### Privacy Statement

This app is designed with no tracking of user data.
